常用的Viewport meta标签

viewport 就是用户网页的可视区域，手机浏览器是把页面放在一个虚拟的”窗口”（viewport）中，通常这个虚拟的”窗口”（viewport）比屏幕宽，这样就不用把每个网页挤到很小的窗口中（这样会破坏没有针对手机浏览器优化的网页的布局），用户可以通过平移和缩放来看网页的不同部分。

基础的viewport meta标签设置使HTML5页面窗口自动调整到设备宽度，并禁止用户缩放页面:

  • width：控制 viewport 的宽度，可以指定的一个具体的值或者特殊的值，如 device-width 为设备的宽度（单位为缩放为 100% 时的 CSS 的像素），可以通过 document.documentElement.clientWidth 获取。
  • height：控制 viewport 的高度，与 width 相对应，可以通过 document.documentElement.clientHeight 获取。
  • initial-scale：初始缩放比例，也即是当页面第一次 load 的时候缩放比例。如果是想要一打开网页就自动以原始比例显示则需要设置为 1.0 。
  • maximum-scale：允许用户缩放到的最大比例。
  • minimum-scale：允许用户缩放到的最小比例。
  • user-scalable：用户是否可以手动缩放，设置为 no 表示不允许用户手动缩放。
